# Initial Conditions Directory

This directory stores the relaxed disk data shared between workflow steps:

- **Input**: Final snapshot from Step 1 (01_relaxation)
- **Output**: Used as `initialConditionsFile` in Step 2 (02_flyby)

## File: `relaxed_disk.csv`

Generated by: `01_relaxation/disk_relaxation` plugin  
Used by: `02_flyby` (razor_thin_hvcc plugins)

**Format**: CSV with columns:
```
id,x,y,z,vx,vy,vz,rho,P,m,u,...
```

**Properties**:
- Hydrostatic equilibrium disk
- Lane-Emden polytropic profile (n=1.5)
- ~10,000 particles (configurable)
- Total mass: 1000 M☉
- Radius: ~3 pc
- Thickness: ~0.4 pc (2.5D)

## Workflow

```
Step 1: Relaxation
   ↓
   Generates: relaxed_disk.csv
   ↓
Step 2: Flyby
   Loads: relaxed_disk.csv
   Adds: IMBH particle
   Runs: Flyby simulation
```

## Notes

- This file is **not version controlled** (large size)
- Must be regenerated by running Step 1
- Required before running Step 2
- Can be reused for multiple Step 2 runs with different parameters
